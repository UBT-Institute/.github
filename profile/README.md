# UBT Institute

Independent research institute focused on Unified Biquaternion Theory (UBT) —
a framework deriving General Relativity and the Standard Model from the
single algebra ℂ⊗ℍ.

[https://ubt-institute.eu](https://ubt-institute.eu)

## Research methodology
UBT is developed using an AI-augmented research pipeline. Mathematical 
derivations and formal verification are performed with AI assistance; 
all results are audited and classified by the principal researcher 
using a strict proof-level system ([L0]/[L1]/[MC]/[OBS]).
